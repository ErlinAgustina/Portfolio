Tutorial Hosting Website di GitHub Pages

  1. Buat Repository GitHub

    Buka GitHub dan buat repository baru dengan nama sesuai keinginanmu.

  2. Upload File Website

    Upload file HTML, CSS, gambar, dan lainnya ke repository tersebut.

  3. Pilih Cabang (Branch)

    Pastikan kamu berada di branch main.

  4. Aktifkan GitHub Pages

    Masuk ke Settings > Pages. Pilih branch main dan folder /root, lalu klik Save.

  5. Akses Website

    Setelah beberapa detik, website kamu akan online di URL seperti https://username.github.io/reponame/

Jika kamu ingin file index.html muncul di awal, pastikan namanya tidak diubah dan berada di root (paling luar) folder.
